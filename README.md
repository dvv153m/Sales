# Sales
Book Sale

Используемые библиотеки и фреймворки:
Dapper, Fluent migrator, AutoMapper, FluentValidation

Запустить Sales.Auth.Api, Sales.Api, Sales.WebUI

Sales.Auth.Api - Сервис аутентификации, у него своя бд. Предполагается этот сервис не виден снаружи.
Sales.WebUI - сайт распродажи книг. Сначала с него идет запрос к сервису аутентификации.
Sales.Api - Сервис распродажи книг, у него своя бд. Снаружи доступа сюда нет. К этому сервису может обращаться только Sales.WebUI. Для этого настроить CORS.
Структура БД в нем позволяет сохранять не только книги но и любые другие товары.

Основные моменты:
Просмотр книг и корзины возможна без аутентификации
Аутентификация через куки
